# ControlService

Only `Administrators` setting in `FileStorage/config.json` can access these APIs.

## DropObjects

Remove objects from local object storage.

## HealthCheck

Get NetmapStatus(`ONLINE` `OFFLINE`) and HealthStatus(`STARTING` `READY` `SHUTTING_DOWN`)

## NetmapSnapshot

Get current netmap.

## SetNetmapStatus

Set offline.
